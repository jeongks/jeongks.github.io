This is a collection of properties and methods that contain information about the browser and computer screen. We can find dimensions of the screen. With the BOM we can create pop ups on window.

Dialogs
There are three ways to create dialogs: alert(), confirm(), and prompt()
Window.alert(): it will show a pop up
Window.confirm(): it will show a pop up with confirm and cancle
Window.prompt(): it will show a pop up with input section
Location
Location property can provide information of URL.
location.href: return full URL
Window.location.href = ‘https://www.byui.edu
Location.protocol: return with which protocol is used for the URL.
Window.location.protocol >> “https:”
Location.host: return the domain address of URL. And port number
Window.location.host >> www.byui.edu and port. ** if the port is 80, it will be omitted since default is port 80.
Location.hostname: return with domain address.
Window.location.hostname >> www.byui.edu
Location.port: return with port number
Window.location.port >> port number
Location.pathname: return with a string of path that follows the domain

Location.search: return a sting that starts with a ‘?’

Location.hash: return a string that starts with ‘#’ ** if there is no ‘#’ in the address. It will return empty string.

Location.origin: return a sting that shows the protocol and domain where the current page originated from.

Followings are the method of the location object.
Reload(): force to reload of the current page.
Assign(): load another resource from a URL.
Replace(): almost same as assign(). Current page will not be recorded in history.
toString(): returns a string containing the whole URL

History
Window.history.go(1); // goes forward 1 page
Window.history.go(0) // reloads the current page
Window.history.go(-1) // goes back 1 page

Window.open() method can open a page with URL.
Open(param, param, param)
1st parameter: URL
2nd parameter: window title
3rd parameter: list of attributes
Close(): close a window.

moveTo(param, param): move screen to the x and y coordinates.
1st parameter: X coordinate
2nd parameter: Y coordinate

Screen
Screen object contains information of the screen of browser.

Cookies
Cookies are small data that are saved locally on a user’s computer to store useful data such as cart in online shopping. However, it is limited to 20 cookies per site. It is limited to 4KB.
Creating cookies
Document.cookie = “name=Kisuk”;
Document.cookie = “occupation=student”;
Document.cookie = “country=Korea”;
Chaging cookie
Document.cookie = “name=Jacob”;
Document.cookie = “country=US”;
